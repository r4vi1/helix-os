# Contributing to HelixOS

We welcome contributions to help build the future of open-source, distributed AI!

## How to Contribute
1.  **Fork the repository** on GitHub.
2.  **Create a feature branch** for your changes.
3.  **Submit a Pull Request (PR)** with a clear description of your work.

## Coding Conventions
*   **Language**: Python 3.10+
*   **Formatting**: Use `black` for code formatting.
*   **Modularity**: Keep components loosely coupled and focused on single responsibilities.

## Open Issues / To-Dos
*   **Improve Speech Pipeline**: Enhance the robustness of the audio capture and Whisper integration.
*   **Containerized Sub-Agent Spec**: Define the Dockerfile and interface for sub-agents.
*   **K3s Operator Draft**: Start working on the Kubernetes operator for managing agents.
*   **Warm Pool Service**: Design the service that keeps essential agents ready for instant response.

Join the discussion and help us shape the architecture!
